教练端使用指南
使用前须知 新教练使用教练端前请先完成 SAVAGE 客户端小程序的注册登录，未登录客户端将无法正常使用教练端功能。
1
日程报备
底部导航「报备」→ 单日 / 批量报备 → 添加报备
  1. 点击底部导航「报备」，选择「单日报备」或「批量报备」
  2. 选择需要报备的日期后，点击右下角「+」按钮添加报备
  3. 选择「可排课」或「不可排课」时间段，确认后提交

2
查看 / 修改报备
报备 → 批量按钮右侧「记录」→ 查看 / 修改
  1. 在报备页面，点击「批量」按钮右侧的「记录」按钮
  2. 进入查看报备记录列表，可查看历史报备内容
  3. 对于支持修改的查看报备记录，点击即可直接修改并重新提交

3
查看日程
底部导航「日程」→ 切换周视图 / 月视图
  1. 打开教练端小程序，默认进入「日程」页面
  2. 点击顶部「周 / 月」按钮，切换周视图或月视图
  3. 左右滑动日期栏，或点击具体日期，查看单日或单次课程安排

4
查看课程详情
日程 → 点击任意课程卡片
  1. 在日程列表中，点击需要查看的课程卡片
  2. 进入课程详情页，查看课程信息与学员预约/签到情况
  3. 下滑查看学员名单，可切换查看「约课」和「签到」学员

5
课后上传合照
日程 → 已完课课程卡片 → 上传合照 / 服务通知消息卡片

方式一：从日程上传

  1. 在日程列表中找到已完课的课程卡片
  2. 点击卡片上的「上传合照」按钮，进入课程详情页
  3. 在「课后返图与学员名单」区域，点击「上传照片」，选择课程合照并提交

方式二：从服务通知上传

  1. 课程结束后，收到小程序发送的「上传合照」服务通知
  2. 点击消息卡片，直接进入对应课程的详情页
  3. 在「课后返图与学员名单」区域，点击「上传照片」，选择课程合照并提交

6
查看个人资料预览
底部导航「工作台」→「个人资料预览」
  1. 点击底部导航「工作台」，进入工作台首页
  2. 点击「个人资料预览」，查看自己的教练信息
  3. 个人资料预览目前仅支持预览，如需修改请联系城市负责人

7
分享日程安排
日程 → 切换视图 → 右上角「分享」
  1. 进入「日程」页面，切换到需要分享的视图（周视图 / 月视图 / 单次课）
  2. 点击右上角「分享」按钮
  3. 生成课程分享海报预览，可保存图片或转发给微信好友 / 朋友圈

提示：分享海报预览可帮助吸引学员预约课程，建议每次排课后及时分享。
8
消息通知
工作台 →「消息通知」/「通知中心」
  1. 在工作台页面点击「消息通知」，进入通知中心查看最新消息
  2. 点击「消息通知管理」，可设置接收通知的类型与方式
  3. 重要通知（如课程提醒、上传合照提醒）将通过微信服务通知推送

9
查看业绩
底部导航「业绩」→ 查看本月 / 上月 / 自定义时间
  1. 点击底部导航「业绩」，进入业绩中心
  2. 默认展示本月业绩中心总览，可切换查看「本月 / 上月」或通过自定义时间筛选
  3. 点击「总收入」进入业绩明细，查看课时费等详细数据
